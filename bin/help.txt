    ##MAIN ARGUMENTS
    #DATA
    --sra [PATH/TO/CONFIG/FILE]                       <default = "bin/sra.txt> Configuration file: must contain one accession number per line. These accessions will be downloaded from the NCBI. By default the pipeline takes as input the six SRAs of the project article stored in a sra.txt file in the bin folder.
    --reference_genome [DOWNLOAD.URL]                 <defaut = "none"> Takes a download URL as a parameter. This option is mandatory to run the mapping (if it is not specified, the pipeline will automatically stop at trimming level).
    --reference_annotation [DOWNLOAD.URL]             <defaut = "none"> Takes a download URL as a parameter. This option is mandatory to run the counting step (if it is not specified, the pipeline will automatically stop at mapping level).
